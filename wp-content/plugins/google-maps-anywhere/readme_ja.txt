== Description ==

各記事、ページ内の任意の場所にGoogleマップを貼り付けるためのプラグインです。WordPress 2.5 以降で動作します。
また、Googleマップだけでなく、ストリートビューも貼り付けることができます。

== Installation ==

1. `/wp-content/plugins/` ディレクトリに `google-maps-anywhere` ディレクトリを作成し、その中にプラグインファイルを格納してください。
　一般的には .zip から展開された googlemaps-anywhere フォルダをそのままアップロードすれば OK です。
2. WordPress の "プラグイン" メニューから "Head Cleaner" を有効化してください。
3. 新規投稿または投稿の編集画面で "Googleマップを追加"ボタンをクリックしてください。

== Frequently Asked Questions ==

= Googleマップの言語を英語に変えるにはどうすればいいですか？ =

1. "Google Maps Anywhere" ver.1.1.0 以降のバージョンに更新してください。
2. WordPress の管理画面から [設定] - [Google Maps Anywhere] を選択してください。
3. "GoogleマップのURL" 欄に "http://www.google.com/maps"と入力して "設定を更新する" をクリックしてください。 (他の言語の場合は? 例：ドイツ語 "http://www.google.de/maps")
