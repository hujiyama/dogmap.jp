== Description ==

Add Google Maps to your posts and pages.(Only for WordPress 2.5+)
This plugin allows you to easily insert Google Maps and Street View into your blog.

== Installation ==

1. Upload the entire `google-maps-anywhere` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Add New Post or Edit Post, and Click "Add Google Maps".

== Frequently Asked Questions ==

= How do I get it to display english when the small map is clicked? =

1. Update to "Google Maps Anywhere" ver.1.1.0 or later.
2. Admin Dashboard [Settings] - [Google Maps Anywhere]
3. "Google Maps URL" to "http://www.google.com/maps", and click "Update options". (other language? ex. Germany "http://www.google.de/maps")
