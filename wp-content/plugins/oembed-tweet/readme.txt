== Description ==

Insert tweeet only input tweet's permalink. You can edit visual design in your theme directory.

This plugin implemented simple template system.

You can edit HTML and Stylesheet in your theme directory.
Copy The following files to your theme directory then edit it!

* oembed_tweet.tpl
* oembed_tweet.css

== Support ==

* [GitHUb](https://github.com/miya0001/oembed-tweet)
* [Plugin Homepage](http://firegoby.jp/wp/oembedtweet)
* [Support](http://wordpress.org/tags/oembed-tweet)

= Some features: =

* You can embed tweet easily thorough oEmbed interface like youtube.
* JSON data from twitter API will cached. So, not affected by trouble of twitter.
* You can edit HTML and CSS in your theme directory.
(oembed_tweet.tpl or oembed_tweet.css copy to your theme directory then edit it!)


== Installation ==

* A plug-in installation screen is displayed on the WordPress admin panel.
* It installs it in `wp-content/plugins`.
* The plug-in is made effective.


== Screenshots ==
