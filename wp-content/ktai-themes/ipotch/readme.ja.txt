=== Ktai Style 用テーマ「iPotch」 ===
配布元URL: http://wppluginsj.sourceforge.jp/ktai_style/themes/
バージョン: 1.2.0
作者: IKEDA Yuriko

== 解説 ==

「iPotch」は、iPhone, iPod touch のテイストを持つ携帯テーマです。iPod, iPod touch での
WordPress ウェブログ表示を iPhoen 風にする「WPtouch」というプラグイン (*) がありますが、その画面表示に似た出力を Ktai Style で再現できます。

(*) http://www.bravenewcode.com/wptouch/

== 必須環境 ==

* Ktai Style 2.0.0 以降
* WordPress 2.6 以降

== インストール ==

配布ファイルを展開して、ipotch フォルダーを、wp-content/ktai-themes ディレクトリーに転送します。もし、wp-content 直下に ktai-themes ディレクトリーがなければ作成してください。
wp-content/wp-content/ktai_style/themes/ ディレクトリーにインストールしてもいいですが、プラグインの自動アップデートを行うと消されてしまうので、ktai-themes へのインストールを推奨します。

== 制限事項 ==

* このテーマで期待した画面表示になるのは、iモードブラウザ2端末 (2009年春以降発売のクッキー対応機種)、EZweb の WAP2.0 対応端末 (A シリーズの大半と W シリーズ以降すべて)、ソフトバンク 3G 端末、各社スマートフォンです。
* ドコモ FOMA のiモードブラウザ1端末では、画面左右の余白がない程度で、ほぼ EZweb 等と遜色ない表示となります。
* ドコモ mova, ソフトバンク PDC 端末、ウィルコム端末では、端末の制約により、背景が灰色っぽい表示になるだけで、かなり貧弱な表示となります。ご了承ください。
